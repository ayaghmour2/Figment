## The following code was used to create the Ultimate Draft databse. This database takes the
## seasonal statistics in each calender year of the game generated draft classes.

## replace the directory with your personal directory where your github is located.

## the console.

## Once the database is loaded, you can create your own queries (or use one of the default
## options as well) to search for players located in one of the tables. Each table consists
## of the single season statline for each amateur player in the given season. Each table is
## named for the year. So for the 1934 stats, the table is called 1934.
